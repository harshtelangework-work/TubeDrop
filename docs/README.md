## ✨ Features

- 🎥 Download YouTube videos
- 🎵 Download audio
- ⚡ Multiple quality options
- 📊 Real-time download progress
- 📂 Choose download location
- 🖥️ Modern desktop interface built with Electron
- 🔧 Powered by **yt-dlp** and **FFmpeg**
- 🚀 Fast and lightweight

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/harshtelangework-work/TubeDrop.git
```

```bash
cd TubeDrop
```

### Install dependencies

```bash
npm install
```

If your project has a separate `server` folder:

```bash
cd server
npm install
```

---

## ▶️ Run the application

```bash
npm start
```
